# دليل إدارة تفعيل وتعطيل الخدمات

## نظرة عامة
تم إضافة ميزة جديدة لإدارة حالة الخدمات في نظام إدارة المغسلة الذكية. يمكن للمدير الآن تفعيل وتعطيل الخدمات بسهولة من خلال واجهة المستخدم.

## المميزات الجديدة

### 1. تبديل حالة الخدمة (Toggle Service Status)
- **تعطيل الخدمة**: عند الضغط على زر "تعطيل" للخدمة النشطة، سيتم إخفاؤها من قائمة الطلبات الجديدة
- **إعادة التفعيل**: عند الضغط على زر "إعادة تفعيل" للخدمة المعطلة، ستعود لقائمة الخدمات المتاحة

### 2. فلترة الخدمات
تم إضافة ثلاثة أزرار فلترة في صفحة الخدمات:

#### جميع الخدمات
- عرض كل الخدمات (النشطة والمعطلة)
- يتم تمييز الخدمات المعطلة بشارة "معطلة" باللون الأحمر

#### النشطة فقط
- عرض الخدمات المتاحة للاستخدام في الطلبات الجديدة فقط
- هذه هي الخدمات التي يمكن للعملاء طلبها

#### المعطلة فقط
- عرض الخدمات المعطلة مؤقتًا
- مفيد لمراجعة الخدمات التي تم تعطيلها والتحقق منها

## كيفية الاستخدام

### لتعطيل خدمة:
1. افتح صفحة "إدارة الأسعار والخدمات"
2. اختر تبويب "الخدمات العامة"
3. اضغط على زر "تعطيل" (الأحمر) بجانب الخدمة المراد تعطيلها
4. أكد العملية في نافذة التأكيد
5. ستظهر رسالة نجاح وسيتم تحديث حالة الخدمة

### لإعادة تفعيل خدمة معطلة:
1. افتح صفحة "إدارة الأسعار والخدمات"
2. اختر تبويب "الخدمات العامة"
3. اختر فلتر "المعطلة فقط" أو "جميع الخدمات" لعرض الخدمات المعطلة
4. ستجد الخدمات المعطلة مميزة بشارة "معطلة"
5. اضغط على زر "إعادة تفعيل" (الأخضر) بجانب الخدمة
6. أكد العملية في نافذة التأكيد
7. ستظهر رسالة نجاح وستعود الخدمة للحالة النشطة

## المظهر البصري

### الخدمة النشطة:
- خلفية عادية
- بدون شارة
- زر "تعطيل" باللون الأحمر

### الخدمة المعطلة:
- خلفية شفافة بنسبة 65%
- شارة "معطلة" باللون الأحمر في الزاوية
- زر "إعادة تفعيل" باللون الأخضر

## متطلبات الصلاحيات
- هذه الميزة متاحة **للمدير فقط**
- العمال والمستخدمون العاديون يمكنهم عرض الخدمات فقط

## التأثير على النظام

### عند تعطيل خدمة:
- ❌ لن تظهر في قائمة الخدمات عند إنشاء طلب جديد
- ✅ الطلبات الحالية التي تستخدم هذه الخدمة لن تتأثر
- ✅ يمكن إعادة تفعيلها في أي وقت

### عند تفعيل خدمة:
- ✅ ستظهر مباشرة في قائمة الخدمات المتاحة
- ✅ يمكن استخدامها في الطلبات الجديدة

## الملفات المعدلة
- `client/src/pages/Services/Services.jsx` - إضافة وظيفة التفعيل/التعطيل والفلترة
- `client/src/pages/Services/Services.css` - تنسيقات الفلاتر
- `server/src/routes/services.js` - يدعم بالفعل تحديث حقل `is_active`

## API Endpoints المستخدمة
```javascript
// تحديث حالة الخدمة
PUT /api/services/:id
Body: {
  ...serviceData,
  is_active: 0 | 1  // 0 = معطل، 1 = نشط
}

// جلب جميع الخدمات (بما فيها المعطلة)
GET /api/services?all=true
```

## ملاحظات مهمة
1. عند تعطيل خدمة، يتم إجراء "حذف ناعم" (Soft Delete) أي أن البيانات لا تُحذف نهائيًا
2. يمكن استرجاع وتفعيل الخدمة في أي وقت
3. الطلبات السابقة التي تحتوي على خدمات معطلة لن تتأثر
4. يُفضل تعطيل الخدمات بدلاً من حذفها للحفاظ على سلامة البيانات التاريخية

## الدعم الفني
في حالة وجود أي مشاكل أو استفسارات، يرجى التواصل مع فريق التطوير.
