# تحسينات نظام الأسعار - Pricing System Improvements

## 📋 نظرة عامة
تم إجراء تحسينات شاملة على نظام عرض وإدارة الأسعار في صفحة "أنواع الملابس والأحجام" لجعلها أكثر وضوحًا واحترافية.

---

## ✨ التحسينات المنفذة

### 1. **إعادة هيكلة جدول الأسعار**

#### **قبل التحسين:**
- الجدول كان يعرض الأسعار بشكل عمودي (صف لكل سعر)
- صعوبة في مقارنة الأسعار بين الأحجام المختلفة
- تكرار في عرض أسماء الأحجام

#### **بعد التحسين:**
- جدول مصفوفة (Matrix) احترافي
- **الصفوف**: الأحجام المختلفة (صغير، وسط، كبير، إلخ)
- **الأعمدة**: الخدمات المختلفة (غسيل عادي، غسيل جاف، كي، إلخ)
- **الخلايا**: السعر المحدد لكل حجم وخدمة

#### مثال على الجدول الجديد:
```
┌─────────┬──────────┬──────────┬──────────┐
│  الحجم  │ غسيل عادي│ غسيل جاف │    كي    │
├─────────┼──────────┼──────────┼──────────┤
│  صغير  │ 5.00 ر.س │ 10.00 ر.س│ 3.00 ر.س │
│  وسط   │ 8.00 ر.س │ 15.00 ر.س│ 5.00 ر.س │
│  كبير  │ 12.00 ر.س│ 20.00 ر.س│ 8.00 ر.س │
└─────────┴──────────┴──────────┴──────────┘
```

## 📁 الملفات المعدلة

### 1. `client/src/pages/Services/Services.jsx`

#### التغييرات:
```javascript
// إضافة دالة ترتيب الأحجام
const sortSizes = (sizes) => {
  const sizeOrder = {
    'صغير': 1, 'small': 1,
    'وسط': 2, 'medium': 2,
    'عادي': 3, 'regular': 3,
    'كبير': 4, 'large': 4,
    'كبير جداً': 5, 'x-large': 5, 'xlarge': 5
  };
  // ... منطق الترتيب
};

// تعديل جدول المعاينة ليكون على شكل Matrix
<table className="price-preview-table">
  <thead>
    <tr>
      <th>الحجم</th>
      {services.filter(s => s.is_active).map(svc => (
        <th key={svc.id}>{svc.name_ar}</th>
      ))}
    </tr>
  </thead>
  <tbody>
    {itemType.sizes.map(sz => (
      <tr key={sz.id}>
        <td className="size-group-cell">{sz.size_name}</td>
        {services.filter(s => s.is_active).map(svc => {
          const priceObj = sz.prices.find(p => p.service_id === svc.id);
          const price = priceObj ? priceObj.price : 0;
          return (
            <td key={svc.id}>{parseFloat(price).toFixed(2)} ر.س</td>
          );
        })}
      </tr>
    ))}
  </tbody>
</table>
```

4. **احترافية**: مظهر منظم وجذاب

### للإدارة:
1. **تحديث سريع**: تعديل جميع الأسعار من شاشة واحدة
2. **تدقيق أسهل**: مراجعة الأسعار بشكل شامل
3. **تجنب الأخطاء**: الهيكل الواضح يقلل من أخطاء الإدخال
4. **مرونة**: إضافة أحجام وخدمات جديدة بسهولة

## 🔧 الاستخدام

### عرض الأسعار:
1. انتقل إلى "إدارة الأسعار والخدمات"
2. اختر تبويب "أنواع الملابس والأحجام"
3. ستجد بطاقة لكل نوع قطعة تحتوي على جدول الأسعار الكامل

### تعديل الأسعار:
1. اضغط على زر "تعديل الأسعار" في البطاقة
2. سيظهر مودال يحتوي على جدول تفاعلي
3. أدخل الأسعار المطلوبة لكل حجم وخدمة
4. احفظ التغييرات

### إضافة أحجام جديدة:
1. في المودال، استخدم حقل الإدخال لإضافة حجم جديد
2. اضغط Enter أو زر "إضافة"
3. سيظهر الحجم الجديد في الجدول تلقائيًا
4. أدخل الأسعار للحجم الجديد
