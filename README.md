# 🚀 Smart Laundry Management System | نظام إدارة المغسلة الذكية

نظام متكامل واحترافي لإدارة المغاسل الذكية (الملابس، السجاد، وغيرها) مبني بأحدث التقنيات ليوفر تجربة مستخدم (UI/UX) سريعة، مستقرة، وجذابة لكل من الإدارة، الكاشير، العمال، والعملاء.

---

## 📸 نظرة عامة على النظام (System Overview)

يتميز النظام بتصميمه البصري العصري المعتمد على **التأثير الزجاجي (Glassmorphism)** والتدرجات اللونية الجذابة مع نمط داكن خفيف مريح للعين، بالإضافة إلى سرعة الانتقال بين الصفحات والاستجابة الكاملة لكافة الشاشات والأجهزة.

---

## ✨ المميزات الرئيسية (Core Features)

* **📊 لوحة تحكم ذكية (Glassmorphic Dashboard):**
  * إحصائيات فورية لطلبات اليوم، الطلبات قيد المعالجة، الطلبات الجاهزة، والمدخرات المالية.
  * رسم بياني منحني تفاعلي للإيرادات (Curved Area Line Chart) مع معالجة ذكية للتواريخ باللغة العربية.
  * ميزة تصدير أحدث الطلبات مباشرة إلى ملف Excel (CSV) بترميز UTF-8 BOM لضمان قراءة النصوص العربية بشكل سليم.
  * تنبيهات فورية وتلقائية للطلبات المتأخرة عن موعد تسليمها المتوقع.

* **🏷️ نظام تتبع القطع الذكي (QR Code Tracking):**
  * توليد وتعيين رمز QR فريد تلقائياً لكل قطعة عند إنشاء الطلب (مثال: `THOBE-0001` للثوب، `SHIRT-0002` للقميص).
  * إمكانية طباعة ملصقات الـ QR بأبعاد دقيقة (`50mm x 30mm`) للصقها على القطع.
  * صفحة مخصصة للعمال لمسح الـ QR (عبر كاميرا الهاتف أو الإدخال اليدوي) لتحديث حالة القطعة تلقائياً في ثوانٍ معدودة.

* **💬 مشاركة الفواتير وتتبع الطلبات تلقائياً عبر الواتساب:**
  * بمجرد حفظ الطلب الجديد، يقوم النظام تلقائياً بتهيئة رابط الفاتورة وتوجيه الكاشير إلى واتساب لإرسال الفاتورة للعميل.
  * أزرار مشاركة يدوية ذكية في تفاصيل الطلب.
  * تهيئة تلقائية لأرقام الهواتف السعودية لتبدأ برمز الدولة (`966`).

* **🌐 بوابة تتبع عامة للعملاء (Customer Portal):**
  * صفحة عامة تسمح للعميل بالاستعلام وتتبع حالة طلباته لحظة بلحظة برقم الطلب أو الجوال دون الحاجة لتسجيل دخول.
  * قراءة المعاملات وتتبعها تلقائياً بمجرد ضغط العميل على رابط الواتساب المرسل إليه.

* **🔔 نظام تنبيهات عائم مخصص (Professional Toast System):**
  * تنبيهات عصرية ومتحركة بدلاً من رسائل المتصفح المزعجة `alert()` لتأكيد العمليات ونجاحها.

* **💳 إدارة مالية دقيقة (Finance & Revenue):**
  * سجل مفصل لكافة حركات الدفع والتحصيل (دفعة مقدمة، متبقي الطلب، سداد كامل).
  * فلاتر متطورة للبحث بطريقة الدفع ونطاقات التواريخ مع حساب الإجماليات بشكل آمن.

## 🛠️ البنية التكنولوجية (Technology Stack)

* **الواجهة الأمامية (Frontend):** React (Vite), Chart.js (الرسوم التفاعلية), Lucide Icons, Vanilla CSS (متغيرات الألوان والتصميم الزجاجي الفاخر).
* **الخلفية (Backend):** Node.js, Express.js (RESTful APIs), JWT Authentication, bcryptjs (تشفير كلمات المرور), Node-QRCode.
* **قاعدة البيانات (Database):** PostgreSQL (مع استخدام Transactions و Triggers لضمان الحساب المالي الدقيق ومزامنة الحالات).

## 🚀 طريقة التشغيل المحلي (Local Quick Start)

لتشغيل النظام بالكامل في بيئة التطوير المحلية بضغطة زر واحدة:

1. تأكد من تشغيل خادم قاعدة البيانات PostgreSQL وضبط متطلبات ملف `.env` في مجلد `server`.
2. انقر نقراً مزدوجاً على الملف المساعد: **`start.bat`** المتواجد في جذر المشروع.
3. سيقوم الملف بتشغيل خادم الباك إند والفرونت إند معاً وفتح المتصفح تلقائياً.

* **العامل (Worker):** `worker@laundry.com` / `worker123` (تتبع وتحديث حالة القطع عبر الـ QR)
